# bm-blocked

Локальный Windows-сервис для работы со списками запрещённых площадок активных кампаний РСЯ в Яндекс Директе. Сервис загружает клиентов, показывает количество заблокированных площадок, проверяет доступность web-площадок и позволяет удалить подтверждённо неработающие адреса из выбранных кампаний. App-площадки распознаются и пропускаются без проверки. Новые версии автоматически скачиваются из официального GitHub-репозитория и устанавливаются после подтверждения пользователя.

## Установка

1. Откройте раздел **Releases** на странице репозитория.
2. Скачайте `bm-blocked.zip` из последнего выпуска.
3. Полностью распакуйте архив в отдельную папку.
4. Запустите `bm-blocked.exe`.
5. Введите общий логин и пароль компании.

Приложение откроет интерфейс в браузере и продолжит работать в системном трее. Для завершения используйте пункт **Выход** в меню значка трея.

Не переносите EXE отдельно от остальных файлов: рядом должны оставаться `server.js`, `index.html`, `auth-config.json` и папка `runtime`.

## Что потребуется

- Windows 10 или Windows 11;
- доступ в интернет;
- OAuth-токен Яндекс Директа;
- права на выбранный рекламный аккаунт;
- логин и пароль компании для входа в сервис.
